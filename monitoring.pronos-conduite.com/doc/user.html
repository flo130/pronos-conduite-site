Utilisateur

Création d'un utilisateur avec sa "home directory"

Création de l'utilisateur : 
[pi@raspberrypi / ]$ sudo useradd -m flo
Modification du mot de passe de l'utilisateur : 
[pi@raspberrypi / ]$ su
Password: 
[pi@raspberrypi / ]$ passwd flo

Ajout de l'utilisateur dans la liste des "sudoers"

Ouvrir le fichier "/etc/sudoers" avec la commande "visudo" : 
[pi@raspberrypi / ]$ sudo visudo -f /etc/sudoers
Ajouer l'utilisateur à la fin du fichier : 
flo ALL=(ALL) NOPASSWD: ALL

Création d'"alias de commandes UNIX" pour l'utilisateur

Ajouter les lignes suivantes dans le fichier "/home/flo/.bashrc" : 
[pi@raspberrypi / ]$ su flo
Password :
[flo@raspberrypi / ]$ vi ~/.bashrc
Ajouter les lignes suivantes pour créer les "alias" de commande "l" et "ll" : 
alias ll='ls -la --color'
alias l='ls -la --color'
